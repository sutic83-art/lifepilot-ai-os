# LifePilot AI — AI Features Roadmap

## 1. AI Daily Planner
Ulazi:
- taskovi
- energija korisnika
- raspoloženje
- kalendar

Izlaz:
- plan dana po blokovima
- prioriteti
- upozorenja na preopterećenje

## 2. AI Habit Coach
- prepoznaje pad streak-a
- predlaže mikro-navike
- šalje dnevne poruke podrške

## 3. AI Mood Analyst
- analizira journal unose
- detektuje stres / umor / fokus
- pravi nedeljni pregled raspoloženja

## 4. AI Personal CFO
- kategorizacija troškova
- budžet po mesecu
- upozorenja na impulsnu kupovinu
- finansijski savet u prostom jeziku

## 5. AI Weekly Review
- sažetak nedelje
- šta je išlo dobro
- šta je blokiralo korisnika
- plan za narednu nedelju

## 6. AI Memory Layer
- pamti korisničke obrasce
- zna kada je korisnik najproduktivniji
- prilagođava savetovanje ličnom stilu

## 7. AI Voice Assistant
- glasovno dodavanje taskova
- jutarnji brief čitanjem naglas
- večernji review glasom

## Preporučeni redosled implementacije
1. Daily Planner
2. Weekly Review
3. Mood Analyst
4. Habit Coach
5. Personal CFO
6. Memory Layer
7. Voice Assistant
